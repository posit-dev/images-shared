# Switch posit-bakery to uv Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace all poetry tooling in `posit-bakery` with `uv` — build system, dependency management, dev tooling, and CI.

**Architecture:** Single-pass migration. Update `pyproject.toml` first (build system + deps), generate the new lockfile and delete the old one, then update the justfile and CI in that order. The existing test suite validates each stage.

**Tech Stack:** uv, hatchling, uv-dynamic-versioning, GitHub Actions (`astral-sh/setup-uv@v5`)

---

### Task 1: Update `pyproject.toml`

**Files:**
- Modify: `posit-bakery/pyproject.toml`

This task rewrites the poetry-specific sections of `pyproject.toml`. No new files. No poetry files are deleted yet — that happens in Task 2 after `uv sync` succeeds.

### Task 5: Update CI (`ci.yml`)

**Files:**
- Modify: `.github/workflows/ci.yml`

There are two jobs to update: `test` and `release`. Both follow the same pattern.

- [ ] **Step 1: Update the `test` job — add fetch options to checkout**

The `test` job's checkout step currently has no options. `uv build` calls `uv-dynamic-versioning` which needs full git history to compute the version. Replace:

```yaml
      - name: Checkout
        uses: actions/checkout@v5
```

With:
```yaml
      - name: Checkout
        uses: actions/checkout@v5
        with:
          fetch-depth: 0
          fetch-tags: true
```

- [ ] **Step 2: Update the `test` job — replace Python/poetry setup with `setup-uv`**

Remove these three steps:
```yaml
      - name: Install poetry
        working-directory: ./posit-bakery
        run: |
          pipx install poetry

      - name: Setup python
        uses: actions/setup-python@v5
        with:
          python-version-file: "posit-bakery/pyproject.toml"
          cache: "poetry"

      - name : Install dependencies
        working-directory: ./posit-bakery
        run: |
          poetry install
```

Replace with:
```yaml
      - name: Setup uv
        uses: astral-sh/setup-uv@v5
        with:
          python-version-file: "posit-bakery/pyproject.toml"

      - name: Install dependencies
        working-directory: ./posit-bakery
        run: |
          uv sync
```

- [ ] **Step 3: Update the `test` job — replace `poetry run pytest` and `poetry build`**

Replace:
```yaml
      - name: Run pytest
        working-directory: ./posit-bakery
        run: |
          GOSS_PATH=${GITHUB_WORKSPACE}/tools/goss \
          DGOSS_PATH=${GITHUB_WORKSPACE}/tools/dgoss \
          poetry run pytest -n auto \
            --dist loadgroup \
            --cov=posit_bakery \
            --cov-report=xml \
            --junit-xml=results.xml \
            test/

      - name: Build
        working-directory: ./posit-bakery
        run: |
          poetry build
```

With:
```yaml
      - name: Run pytest
        working-directory: ./posit-bakery
        run: |
          GOSS_PATH=${GITHUB_WORKSPACE}/tools/goss \
          DGOSS_PATH=${GITHUB_WORKSPACE}/tools/dgoss \
          uv run pytest -n auto \
            --dist loadgroup \
            --cov=posit_bakery \
            --cov-report=xml \

        working-directory: ./posit-bakery
```
