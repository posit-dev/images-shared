# bakery get

Get information about the bakery configuration

``` bash
bakery get COMMAND [ARGS]...
```

    Usage: bakery get [OPTIONS] COMMAND [ARGS]...

      Get information about the bakery configuration

    Options:
      --help  Show this message and exit.

    Commands:
      tags     Get the list of tags that would be built for the given context...
      version  Get a version name from bakery.yaml.

## Commands
