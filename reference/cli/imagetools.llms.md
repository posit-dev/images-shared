# bakery imagetools

Merge and SOCI-convert multi-platform images (ORAS + SOCI)

``` bash
bakery imagetools COMMAND [ARGS]...
```

    Usage: bakery imagetools [OPTIONS] COMMAND [ARGS]...

      Merge and SOCI-convert multi-platform images (ORAS + SOCI)

    Options:
      --help  Show this message and exit.

    Commands:
      merge         Merge multi-platform images from build metadata files...
      soci-convert  Convert images referenced by build-metadata JSON files...

## Commands

`merge`  
[Merge multi-platform images from build metadata files using ORAS.](../../reference/cli/imagetools/merge.llms.md)

`soci-convert`  
[Convert images referenced by build-metadata JSON files into SOCI-enabled images.](../../reference/cli/imagetools/soci_convert.llms.md)
