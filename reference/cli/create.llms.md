# bakery create

Create new projects, images, and versions (aliases: c, new)

``` bash
bakery create COMMAND [ARGS]...
```

    Usage: bakery create [OPTIONS] COMMAND [ARGS]...

      Create new projects, images, and versions (aliases: c, new)

    Options:
      --help  Show this message and exit.

    Commands:
      project  Creates a new project in the context path
      image    Creates a quickstart skeleton for a new image in the context path
      version  Renders templates for an image to a versioned subdirectory of...
      matrix   Renders templates for an image to a single subpath of the...

## Commands

`project`  
[Creates a new project in the context path](../../reference/cli/create/project.llms.md)

`image`  
[Creates a quickstart skeleton for a new image in the context path](../../reference/cli/create/image.llms.md)

`version`  
[Renders templates for an image to a versioned subdirectory of the image directory.](../../reference/cli/create/version.llms.md)

`matrix`  
[Renders templates for an image to a single subpath of the image directory using a version matrix.](../../reference/cli/create/matrix.llms.md)
