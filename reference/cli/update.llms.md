# bakery update

Update managed files and configurations (aliases: u, up)

``` bash
bakery update COMMAND [ARGS]...
```

    Usage: bakery update [OPTIONS] COMMAND [ARGS]...

      Update managed files and configurations (aliases: u, up)

    Options:
      --help  Show this message and exit.

    Commands:
      files    Renders templates to version files matching the given filters
      version  Update an existing image version with a new version number.
