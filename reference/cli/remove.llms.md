# bakery remove

Remove images and versions from the project (aliases: rm, r)

``` bash
bakery remove COMMAND [ARGS]...
```

    Usage: bakery remove [OPTIONS] COMMAND [ARGS]...

      Remove images and versions from the project (aliases: rm, r)

    Options:
      --help  Show this message and exit.

    Commands:
      image    Removes an existing image from the bakery project
      version  Removes an existing version from an image in the bakery project

## Commands

`image`  
[Removes an existing image from the bakery project](../../reference/cli/remove/image.llms.md)

`version`  
[Removes an existing version from an image in the bakery project](../../reference/cli/remove/version.llms.md)
