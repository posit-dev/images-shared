# bakery clean

Cleaning utilities for remote build caches

``` bash
bakery clean COMMAND [ARGS]...
```

    Usage: bakery clean [OPTIONS] COMMAND [ARGS]...

      Cleaning utilities for remote build caches

    Options:
      --help  Show this message and exit.

    Commands:
      cache-registry  Cleans up dangling caches in an external registry
      temp-registry   Cleans up temporary images in an external registry
